# Vertex

## 三角形のプレイヤーがバインドの能力を使い天井を目指す。
基本移動はWASD

マウスを左クリックすることで **バインド(Bind)** の能力を発動できる

道中にある消える四角形を取ると**ポイント**をゲット（未実装）

壁に当たると三角形の体なので視点が**傾いてしまう**のに気を付けよう

**タイム**を縮めつつ（未実装）、**高得点**を狙おう
